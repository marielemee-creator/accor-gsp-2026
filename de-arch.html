⚙️ Notes techniques — pour l'équipe ingénierie
Déploiement widget chat
SF Messaging for Web : 1 snippet JavaScript sur les pages Accor web/app. Remplacement du widget existant — aucun backend custom requis.
Real-Time Translation
Natif dans SF Digital Engagement, sans provider tiers. Config dans Setup → Digital Engagement → Translation Settings. 20+ langues supportées OOB.
Profil ALL — enrichissement contexte
Données ALL (segment, historique réservations) accessible via Data Cloud connector ou SOQL lookup sur Contact. Apparaît dans la console à l'ouverture du chat.
Agentforce bot — topics et déflexion
Bot natif dans Omnichannel — pas d'intégration webhook. Topics configurables (info hôtel, statut réservation, FAQ). Handoff automatique avec contexte si bot escalade.
