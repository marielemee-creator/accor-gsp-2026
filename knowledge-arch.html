⚙️ Notes techniques — pour l'équipe ingénierie
Structure articles SF Knowledge
Articles organisés en Data Categories (marque, sujet, langue). Workflow publication : Draft → Review → Published. Versioning natif. Multi-langue avec articles liés.
Agent Assist — surfacing temps réel
Einstein analyse le transcript en cours et suggère les articles Knowledge correspondants. Activer dans Setup → Einstein for Service → Agent Assist. Aucun ML training requis.
BYOM — Knowledge Retrieval action
Action Agentforce native : Search Knowledge. Récupère les articles pertinents → injecte dans le prompt AccorGPT. Filtrage par Data Category possible pour cibler par marque/sujet.
Migration Mayday → SF Knowledge (ARI)
Export Mayday → import via SF Data Loader ou API bulk. Mapping des catégories Mayday vers Data Categories SF. Validation éditoriale avant publication. ARI Pilot : progression en cours depuis juin 2026.
