La migration téléphonique prévue pour fév. 2028 n'est pas bloquante : SF Voice fonctionne dès aujourd'hui avec l'infra Amazon Connect existante.
⚙️ Notes techniques — pour l'équipe ingénierie
Transcription — Contact Lens API
Contact Lens for Amazon Connect fournit le transcript en streaming. SF reçoit via Voice Call Transcript object — stocké dans SF, consultable dans le Case.
Work Summary — Einstein AI
Modèle SF natif entraîné sur contact center data. Option BYOM possible (AccorGPT) pour la génération du résumé — à confirmer sur roadmap SF BYOM.
Agent Assist — Knowledge grounding
Suggestions basées sur SF Knowledge articles + transcript en cours. Aucune configuration ML requise — activer dans Setup → Einstein for Service.
Supervisor Console — unified view
Fusionne Omnichannel Supervisor (digital) et Voice Supervisor. Whisper coaching, barge, monitoring sentiment en temps réel via Contact Lens sentiment score.
